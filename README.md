# KUL_neuroimaging_tools
Tools to analyse (resting/task-based) fMRI, diffusion & structural MRI data

Project PI's: Stefan Sunaert

Requires Mrtrix3, FSL, ants, dcm2niix, dcm2bids (jooh fork), docker, fmriprep and mriqc
